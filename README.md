# kanghui-learning.github.io

Personal academic website of Kanghui Ning, built with [al-folio](https://github.com/alshedivat/al-folio) (Jekyll).

- Local preview: `docker compose up` then open http://localhost:8080
- Deployed to GitHub Pages via `.github/workflows/deploy.yml` (builds to the `gh-pages` branch)
- Publications live in `_bibliography/papers.bib`; news in `_news/`; CV data in `_data/cv.yml`
